# Punar Bhoj Restaurant Website

Punar Bhoj is an online restaurant experience that revives and celebrates old, lost Indian recipes and culinary traditions. This project showcases unique dishes rooted in heritage, designed to create nostalgia and connect people with forgotten flavors.



## Features

Discover and explore ancient Indian recipes.



Visual storytelling with images and history.



Responsive design for all devices.



## Getting Started

Follow these steps to set up and run the website locally from a ZIP file:



1\. Download and Extract the ZIP File

Locate the ZIP file sent to you via email (usually named punarbhoj.zip or similar).



Right-click on the ZIP file and select Extract All... (Windows) or use Archive Utility (Mac) to extract the contents into a folder (e.g., PunarBhoj).



2\. Open the Project Folder

Navigate to the extracted project folder.



You should see all the website files, including index.html, images, stylesheets, and scripts.



3\. Run the Website Locally

Locate the index.html file inside the project folder.



Double-click on index.html to open it in your default web browser.



Alternatively, you can right-click the file and choose Open with... to select your preferred browser.



This will display the full Punar Bhoj website on your local machine.
